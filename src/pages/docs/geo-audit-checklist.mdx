## GEO Audit Checklist v1

Use this checklist to launch a focused GEO audit in one week.

### 1. Baseline and Scope

- Define target entities and core topics.
- Record current visibility across AI engines.
- Assign owners for content, technical markup, and reporting.

### 2. Content Readiness

- Identify pages that should become citation sources.
- Add explicit entity naming and concise factual sections.
- Ensure each page has one clear artifact output.

### 3. Technical Readiness

- Validate JSON-LD and core schema coverage.
- Ensure markdown structure is machine-readable.
- Check canonical, robots, and indexing directives.

### 4. Measurement Setup

- Define weekly GEO visibility indicators.
- Establish citation and entity-coverage reporting cadence.
- Track changes by publish date and revision owner.

### 5. First-Week Actions

- Publish one revised execution artifact.
- Publish one standards page with metadata examples.
- Review early retrieval and citation signal changes.
